         <p class="subtitle">Git</p>
         <ul>
            <li><b>git init</b> — создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий.</li>
            <li><b>git clone &lt;URL&gt;</b> — клонирование репозитория из удаленного хранилища на локальную машину.</li>
            <li><b>git push</b> — выгрузка сделанных изменений в удаленный репозиторий (например, GitHub).</li>
            <li><b>git pull</b> — получение изменений из удаленного репозитория на локальную машину.</li>
            <li><b>git status</b> — просмотр списка измененных файлов.</li>
            <li><b>git branch название_ветки</b> — создание ветки.</li>
            <li><b>git checkout название_ветки</b> — переход на ветку.</li>
            <li><b>git commit</b> — сохранение изменений.</li>
            <li><b>git log</b> — просмотр истории коммитов в репозитории.</li>
